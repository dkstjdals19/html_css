HTML

html은 HTML(HyperText Markup Language)은 웹 페이지를 만드는 데 사용하는 언어야. 쉽게 말하면, 웹사이트의 뼈대를 만드는 역할을 하지!




CSS

Css는 CSS는 HTML로 만든 웹페이지의 디자인(스타일)을 꾸미는 언어야! HTML이 웹사이트의 **뼈대(구조)**라면, CSS는 색상, 크기, 위치, 애니메이션 등을 조절해서 더 예쁘고 보기 좋게 만들어 줘.




JAVASCRIPT

javascript는 JavaScript는 웹사이트에 다양한 기능과 인터랙션을 추가하는 프로그래밍 언어야! HTML이 웹사이트의 뼈대(구조), CSS가 **디자인(스타일)**을 담당한다면, JavaScript는 웹페이지를 "동적으로" 만들고, 사용자와 상호작용할 수 있도록 해 줘!
